# Music Vault UI icons

These SVG files are original geometric interface assets created for Music Vault.
They are distributed under the repository's license and are not copied from
Spotify or any third-party icon library.

The icons use a consistent 24 by 24 view box, rounded strokes, and no embedded
text, fonts, scripts, raster images, personal data, or external resources. The
application renders and tints them at runtime for normal, active, and disabled
states.
