# Music Vault v1.0.0 — first run

1. Extract the complete portable folder to a writable location.
2. Run `MusicVault.exe` from that folder.
3. Follow the first-run guide. Local-file import works without an API key or
   FFmpeg.
4. Configure FFmpeg/ffprobe separately only if you plan to synchronize and
   convert authorized public or unlisted YouTube playlists.

Music Vault creates a private `data` folder beside the executable unless you
explicitly choose another writable runtime location. The release contains no
music, database, configuration, status file, API key, or FFmpeg command-line
tools.

Use only content you are authorized to download. See `AUTHORIZED_USE.md` and
`BINARY_DISTRIBUTION_LICENSE.md` for the product and distribution boundaries.
